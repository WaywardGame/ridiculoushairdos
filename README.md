# Ridiculous Hairdos
Because normal hairdos are boring! 

An example mod to showcase the functionality of the customizations.json format.

## Cloning:
```
git clone https://github.com/WaywardGame/ridiculoushairdos.git
cd ridiculoushairdos
```
